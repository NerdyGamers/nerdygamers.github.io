FAQ

What game are these plugins for?
Unturned, Ultima Online, andRust. If more games are requested and we have the ability, we might add them to the list.
Unturned Plugins
These Plugins are written for the RocketMod plugin framework.
How do I install Unturned Plugins?
Drop the Plugin.dll into the 'Unturned\Servers\UnturnedServer\Rocket\Plugins' directory. If your server is running, it will compile and enable the new plugin automaticly.
Ultima Online Server Scripts
Ultima Online/'UO' Scripts, were coded using the ServUO FreeServer
OrionUO Client, GitHub, Discord, Docs, Macros/Scripts
How can I install the Ultima Online Scripts?
Place the single Script.cs or the Script System Folder that contains the '.CS' files into the 'UOServer\Scripts\Custom' directory. Before your new script is usable ingame you must ShutDown the Server and Restart it to compile the code.
Rust Plugins
The Rust Plugins were created with the Oxide plugin framework.
How do I install the Rust plugins?
If you have RocketMod installed in your server, Drag and drop the Plugin.DLL into the 'RustServer\Oxide\Plugins' directory. If your server is running, it will compile and enable the new plugin automaticly.
